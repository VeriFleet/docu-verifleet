VeriFleet Help

Entdecken Sie die Möglichkeiten

VeriFleet ist eine digitale Führerscheinkontrolle für ihren Fuhrpark. Verifleet basiert auf einem KI gestützten Prüfalgorithmus, um Ihnen höchste Sicherheit bei der Führerscheinkontrolle Ihrer Mitarbeiter zu bieten.

Optionen

Alle Optionen, die VeriFleet Ihnen bietet, sind digital implementiert, so dass Sie jederzeit einen vollständigen Überblick über alle Prüfungsrelevanten Details Ihrer Fahrer haben.

Kartenführerscheine

VeriFleet prüft Kartenführerscheine mithilfe einer selbstlernenden KI (Künstliche Intelligenz). Dadurch kann eine Kontrolle sehr sicher durchgeführt werden. Es werden alle relevanten Merkmale des Führerscheins geprüft, auch Hologramme. Deshalb haben Sie mit VeriFleet ein Höchstmaß an Sicherheit.

Die selbstlernende KI gewährleistet auch eine schnelle und sichere Erkennung des Führerscheins. Ihre Fahrer müssen also nur sehr wenig Zeit für die Kontrolle aufwenden und Sie sparen sich unnötige Rückfragen.

Ältere Führerscheine (Siegelkontrolle)

VeriFleet prüft altere Führerschein mithilfe von Siegeln. Sie können diese Siegel ganz einfach auf die Führerscheine aufkleben. Das Siegel ist im System mit dem Datensatz des Fahrers verknüpft, so dass eine eindeutige und sichere Kontrolle möglich ist. Ein weiteres Sicherheitsmerkmal besteht darin, dass ein Ablösen des Siegels nur bei gleichzeitigem Zerstören des Siegels möglich ist. So wird verhindert, dass ein Siegel auf ein anderes Dokument aufgebracht wird.

UVV

Mit VeriFleet können sie ebenfalls die UVV rechtssicher durch den Fahrer durchführen lassen. Damit decken Sie alle gesetzlichen Anforderungen zur Kontrolle Ihrer Fahrer um und minimieren Ihr Haftungsrisiko.

Vorteile

kein Fahrer-Login erforderlich

Keine App, kein Login, damit wollen wir vermeiden, dass Fahrer Passwörter merken müssen oder den Fuhrparkleiter kontaktieren müssen, um Passwörter zurückzusetzen.

Aber wie funktioniert es dann?

Ganz einfach, der Fahrer erhält vom System automatisch eine Nachricht vom System, in der ein Link enthalten
ist, mit dem er die Prüfroutine starten kann. Dieser Link ist eindeutig und kann nur für die angeforderte Prüfung verwendet werden.

Die Prüfaufforderung wird per Mail oder per SMS an den Fahrer gesendet, damit können Sie auch Fahrer erreichen, die keine individuelle Mailadresse besitzen.

Prüfung durch den Fuhrpark-Verantwortlichen

Alternativ kann ein Fahrer im Systemn so angelegt werden, dass die Prüfungsanforderung an den Fuhrpark-Verantwortlichen gesendet wird. So können sie sicherstellen, dass eine Prüfung des Führerscheins durch sie persönlich erfolgt. Ebenso können auf diese Weise Führerscheine von Fahrern geprüft werden, die kein Smartphone besitzen.

Überwachung angeforderter Prüfungen

VeriFleet überwacht, ob für alle Fahrer gültige Prüfungen vorliegen und zeigt Ihnen dies auf dem DashBoard an. Darüber hinaus werden sowohl Fahrer als auch Sie selbst bei überfälligen Prüfungen automatisch vom System informiert bzw. erneut zur Prüfung aufgefordert (Eskalations-Management).
